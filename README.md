# TP Tours de Hanoï

## Prérequis

- python 3.11 (devrait fonctionner avec python 3.10)
- jupyter notebook (notebook)
- pygame

## Lancement

```console
jupyter notebook
```

Une fois jupyter lancé dans le navigateur, ouvrir 'Hanoi_JPO.ipynb' puis réaliser le TP.

## Reset

- Fermer les fenêtres Jupyter dans le navigateur et arrêter le serveur Jupyter (avec Ctrl+C).
- Exécuter le script `reset.sh`
